# Завдання 10: Перевірка числа на простоту

**Рівень:** Medium  
**Тема:** Do-While Loop  

**Опис:**  
Створи цикл `do...while`, який перевіряє, чи є число n простим. Виведи true або false.

---

## 📥 Вхідні дані
- n (ціле число, n > 1)

## 📤 Вихідні дані
- true, якщо число просте; false — якщо ні
