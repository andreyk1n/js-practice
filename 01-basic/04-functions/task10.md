# Завдання 10: Функція для повторення рядка

**Рівень:** Medium  
**Тема:** Functions  

**Опис:**  
Створи функцію `repeatString`, яка приймає рядок s і число n і повертає рядок, повторений n разів.

---

## 📥 Вхідні дані
- s (рядок), n (ціле число >= 0)

## 📤 Вихідні дані
- Рядок s повторений n разів
